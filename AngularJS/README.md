### Abrir consola como administrador
```
ng -v
```

### Navegar a la carpeta deseada y crear esqueleto del proyecto

```
cd C:/..
ng new my-app
```

### Iniciar servidor local
http://localhost:4200
```
ng serve -open
```

#Instalar dependencias
```
npm install bootstrap@4.0.0-beta.3

```
